Запуск:
Для запуска проекта необходим Python версии 3 и выше, а так же библиотеки, указанные в requirements.txt
Приложение запускается из командной строки командой python setup.py
Для примеров запросов, описанных ниже, необходим установленный cURL

Файлы:
requirements.txt - названия требуемых для запуска api библиотек
setup.html - документация
app.log - файл регистрации

База данных:
В качестве БД был взят MySQl. БД находится на серверах AWS.
Доступные пользователи для проверки уникальности: Test1 и Test2.
Пароли для пользователей не были созданы, т.к. этого условия нет в задании.

Примеры запросов:
1) Редактировать выбранную шутку
curl -i -u Test1:python -X PUT -H "Content-Type: application/json" -d "{"""text""":"""измененный текст шутки"""}" http://127.0.0.1:5000/todo/api/jokes/<id шутки>
2) Сгенерировать шутку (если полученная шутка уже есть в бд, она не записывается)
curl -i -u Test1:python http://127.0.0.1:5000/todo/api/jokes/new
3) Посмотреть список всех шуток
curl -i -u Test1:python http://127.0.0.1:5000/todo/api/jokes
4) Посмотреть шутку по Id (показывает только шутки, принадлежащие пользователю)
curl -i -u Test1:python http://127.0.0.1:5000/todo/api/jokes/<id шутки>
5) Удалить шутку
curl -i -u Test1:python -X "DELETE" http://127.0.0.1:5000/todo/api/jokes/<id шутки>
